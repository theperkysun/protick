# protick

A lightweight Node.js app for running the Protick interface locally.

## Prerequisites

- [Node.js](https://nodejs.org/) 18+ (or a recent LTS version)

## Install

From the project root, install dependencies (if any are added later):

```bash
npm install
```

## Run the app

Start the server:

```bash
npm start
```

For development with automatic restart on file changes:

```bash
npm run dev
```

Then open your browser to:

- <http://localhost:3000>

## Project structure

- `server.js` – HTTP server entry point
- `public/` – static frontend assets
- `stitch/` – design references and exported screens
